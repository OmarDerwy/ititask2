# Omar Derwy - Day2 Task
## How to delete branch locally and remotely
```bash
git push origin --delete dev
git push origin --delete test
git branch -d dev
git branch -d test
```

## how to list tags
```bash
git tag
```
## to delete tag remotely and locally
```bash
git push origin --delete v1.7
git tag -d 1.7
```
